Technical Support Engineer Assessment

This project is a simple checkout simulation to demonstrate understanding
of order creation, checkout flow, and payment process.

Tech Stack:
- PHP
- MySQL
- HTML

Flow:
1. User creates an order.
2. Order is saved in the database.
3. User proceeds to checkout.
4. Payment is simulated as successful.
5. Order and payment status are stored.

How to run:
- Place the project folder inside xampp/htdocs
- Start Apache and MySQL
- Import database.sql using phpMyAdmin
- Open http://localhost/checkout/create_order.php
